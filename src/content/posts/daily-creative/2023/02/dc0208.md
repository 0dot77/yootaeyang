# Main question

## 당신은 지금 어떤 무한 게임에 참여하고 있는가?

🔍 현재 나에게 주어진 무한 게임은 앞으로 묵묵히 나아가는 것이다. 아직은 어떤 것에 있어서도 '전문가'의 위치에 도달했다고 생각하지 않는다. 꼭 하나의 일에만 전문가가 될 필요는 없겠지만, 어느정도의 숙련도는 갖춘 사람이 되고 싶다.
